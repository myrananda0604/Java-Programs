# Java Programs 💻

This repository contains 22 Java programs covering various core concepts of Java programming.

## 📚 Topics Covered

* Basic Programs (Arithmetic, Factorial, etc.)
* Arrays
* Strings
* Object-Oriented Programming (OOP)
* Multithreading
* GUI using Swing
* Exception Handling
* File Handling
* Packages and Subpackages

## 🛠️ Technologies Used

* Java
* Swing (for GUI programs)

## ▶️ How to Run

1. Compile the program:
   javac FileName.java

2. Run the program:
   java FileName

## 📁 Repository Structure

* Each program is saved as a separate `.java` file
* Package-related programs are organized in folders
